Modo de Uso:

Para utilizar o software e executar o algoritmo de Tarjan, siga as seguintes etapas:

1. Adicionar Vértices:

Clique no botão "Adicionar Vértice" para criar um novo vértice. Será solicitado que você informe a quais outros vértices este vértice irá apontar. Digite os números correspondentes aos vértices desejados e pressione a tecla "Enter". Para finalizar a adição de vértices, insira um caractere não numérico. Para adicionar mais vértices, clique novamente no botão, e o ID do vértice será incrementado automaticamente. Você também pode usar a tecla "Espaço" como atalho para esse botão.

2. Botão "Deletar Vértice":

Clique no botão "Deletar Vértice" e digite o ID do vértice que você deseja deletar. Se o vértice existir, ele será removido, caso contrário, você receberá uma mensagem informando que não há um vértice com o ID especificado.

3. Botão "Algoritmo de Tarjan":

Ao clicar no botão "Algoritmo de Tarjan", o algoritmo será executado com os vértices criados até o momento.

Para reiniciar o código, basta recarregar a página (tecla: F5).
